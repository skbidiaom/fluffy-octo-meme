# 🎮 R6 Clone Game

This is a fast-paced tactical shooter inspired by classic R6-style gameplay. Built with a focus on smooth multiplayer, clean mechanics, and fun hidden details.

---

### 🌵 Link For Game
The link for the game is:
r6-clone-game.vercel.app

### 🔫 Gameplay

* Tactical shooter experience with R6-style mechanics
* Smooth movement and responsive controls
* Multiple weapons including guns and grenades

### 🌐 Multiplayer System

* Invite-code based multiplayer system
* Easily join friends in private sessions
* Real-time gameplay with synced actions

### 🎵 Background Music

* Immersive background music while playing
* Enhances atmosphere and gameplay intensity

### 🛠️ Admin Panel

* Built-in admin controls
* Manage players and game settings
* Useful for testing and private matches

### 🛒 Shop System

* Buy weapons and grenades
* Expand your loadout
* Progress through gameplay

### 🥚 Easter Eggs & References

* Hidden secrets across the game
* References to other games and media
* Explore and discover them all
